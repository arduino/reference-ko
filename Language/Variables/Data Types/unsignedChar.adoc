= unsigned char


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
unsigned data 자료형은 1바이트 메모리를 차지한다. link:../byte[byte] 자료형과 같다.

unsigned char 자료형은 0 에서 255 까지 숫자를 인코딩한다.

아두이노 프로그램 일관성을 위해, byte 자료형이 선호된다.
