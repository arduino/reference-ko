= double


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
배밀도 부동소수점 숫자. 우노와 ATMEGA 기반 보드에서, 이것은 4바이트를 차지한다.
즉, double 구현은 float 와 정확히 같고, 정확도에서 얻는 게 없다.
아두이노 Due에서, double 은 8바이트 (64비트) 정밀도를 가진다.

[%hardbreaks]
