[float]
=== 설명
short 는 16비트 자료형이다.
모든 아두이노(ATMega 와 ARM 기반)에서 short 는 16비트 (2바이트) 값을 저장한다.
-32,768 에서 32,767 (최소값 -2^15 그리고 최대값 (2^15) - 1)
[%hardbreaks]

[float]
=== 문법
`short var = val;`

`var` - short 변수 이름
`val` - 변수에 할당할 값
