= trim()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
앞뒤 공백을 없앤 스트링 버전을 얻는다. 1.0부터 trim ()은 새로운 스트링을 반환하지 않고 스트링을 바꾼다.
[%hardbreaks]


[float]
=== 문법
`myString.trim()`

[float]
=== 매개변수
`myString`: 스트링 형 변수


[float]
=== 반환
없음
