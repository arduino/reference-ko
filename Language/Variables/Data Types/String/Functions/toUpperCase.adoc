= toUpperCase()


// OVERVIEW SECTION STARTS
[#overview]

[float]
=== 설명
대문자 버전 스트링을 얻는다. 1.0 부터, toUpperCase() 는 새 스트링을 반환하는 대신 스트링을 바꾼다.
[%hardbreaks]


[float]
=== 문법
`myString.toUpperCase()`

[float]
=== 매개변수
`myString`: 스트링 형 변수


[float]
=== 반환
없음
