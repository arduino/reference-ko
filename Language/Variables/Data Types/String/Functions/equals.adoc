= equals()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
두 스트링이 같은지 비교. 대소문자 구별, 즉 "hello"는 "HELLO"와 같지 않다.
[%hardbreaks]


[float]
=== 문법
`myString.equals(myString2)`

[float]
=== 매개변수
`myString, myString2`: 스트링 형 변수


[float]
=== 반환
`true`: myString 이 myString2 와 같으면

`false`: 아니면
