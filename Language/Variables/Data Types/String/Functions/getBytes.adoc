= getBytes()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
스트링의 문자를 제공된 버퍼에 복사.
[%hardbreaks]


[float]
=== 문법
`myString.getBytes(buf, len)`

[float]
=== 매개변수
`myString`: 스트링 형 변수

`buf`: 문자를 복사해 넣을 버퍼(_byte []_)

`len`: 버퍼의 크기 (_unsigned int_)

[float]
=== 반환
없음
