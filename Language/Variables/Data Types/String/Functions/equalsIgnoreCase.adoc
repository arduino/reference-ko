= equalsIgnoreCase()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
두 String이 같은지 비교. 대소문자 구별 안 하며, String("hello") 가 String("HELLO")와 같음을 뜻함.
[%hardbreaks]


[float]
=== 문법
[source,arduino]
----
string.equalsIgnoreCase(string2)
----

[float]
=== 매개변수
`string, string2`: 스트링 형 변수


[float]
=== 반환
`true`: string 과 string2 이 같으면(대소문자 무시)

`false`: 아니면
