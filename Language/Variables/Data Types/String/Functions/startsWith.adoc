= startsWith()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
스트링이 다른 스트링의 문자들로 시작하는지 아닌지 테스트

[%hardbreaks]


[float]
=== 문법
`myString.startsWith(myString2)`

[float]
=== 매개변수
`myString, myString2`: a variable of type String


[float]
=== 반환
`true`: myString 이 myString2의 문자들로 시작하면

`false`: 아니면
