[float]
=== 설명
String의 문자를 제공되는 버퍼에 복사.
[%hardbreaks]


[float]
=== 문법
[source,arduino]
----
string.toCharArray(buf, len)
----

[float]
=== 매개변수
`string`: 스트링 형 변수

`buf`: 문자를 복사해 넣을 버퍼 (_char []_)

`len`: 버퍼 크기 (_unsigned int_)

=== 반환
