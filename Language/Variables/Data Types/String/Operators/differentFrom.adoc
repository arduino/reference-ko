= != 다름


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
두 스트링이 다른지 비교. 비교는 대소문자 구별하며, 이는 "hello" 와 "HELLO" 가 다름을 뜻한다. 기능적으로 string.equals() 와 같다.
[%hardbreaks]


[float]
=== 문법
[source,arduino]
----
myString1 != myString2
----

[float]
=== 매개변수
`myString1, myString2` - String 변수

[float]
=== 반환
`true`: myString1 이 myString2 와 다르면

`false`: 아니면
