= == 비교


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
두 스트링이 같은지 비교한다. 결과는 대소문자 구별하며, 이는 "hello"는 "HELLO"와 같지 않다는 것을 뜻한다. 기능적으로 string.equals()와 같다.



[%hardbreaks]


[float]
=== 문법
[source,arduino]
----
myString1 == myString2
----

[float]
=== 매개변수
`myString1, myString2` - String 변수

[float]
=== 반환
스트링의 n 번째 문자. charAt() 과 같다.

`true`: myString1 이 myString2 와 같으면

`false`: 아니면
