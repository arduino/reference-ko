= [] Element Access


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
String의 개별 문자에 접근하는 것을 허용.

[%hardbreaks]


[float]
=== 문법
[source,arduino]
----
char thisChar = string1[n]
----

[float]
=== 매개변수
`char thisChar` - 문자 변수

`string1` - String 변수

`int n` - 숫자 변수

[float]
=== 반환
String의  n번째 char. Same as charAt().
스트링의 n번째 문자. charAt()와 같음.
