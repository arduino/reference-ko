= word()


// OVERVIEW SECTION STARTS
[#overview]

[float]
=== 설명
값을 link:../../data-types/word[word] 자료형으로 변환.

[%hardbreaks]


[float]
=== 문법
`word(x)` +
`word(h, l)`

[float]
=== 매개변수
`x`: 값

`h`: high-order (가장 왼쪽) 바이트

`l`: low-order (가장 오른쪽) 바이트
[float]
=== 반환
`word`
