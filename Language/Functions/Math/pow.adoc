= pow(base, exponent)

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자 거듭제곱 값 계산. `pow()` 는 숫자를 분수 제곱하는 데 쓸 수 있다. 이것은 곡선의 지수 매핑을 생성할 때 쓸모 있다.

[%hardbreaks]


[float]
=== 문법
`pow(base, exponent)`


[float]
=== 매개변수
`base`: 숫자 (`float`)

`exponent`: base 를 거듭 곱할 횟수 (`float`)

[float]
=== 반환
거듭제곱 결과. (`double`)
