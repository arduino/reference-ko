= abs(x)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자의 절대값 계산.

[%hardbreaks]


[float]
=== 문법
`abs(x)`

[float]
=== 매개변수
`x`: 숫자

[float]
=== 반환
`x`: x 가 0보다 크면.

`-x`: if x 0보다 작으면.

[float]
=== 주의와 경고
abs() 함수가 구현된 방법 때문에, 괄호 안에 다른 함수를 쓰는 것을 피하세요. 잘못된 결과가 나올 수 있음.

[source,arduino]
----
abs(a++);   // 이렇게 하지 마세요 - 잘못된 결과 나옴

abs(a);         // 대신 이것을 사용하세요 -
a++;        // 함수 밖에서 다른 수학
----
[%hardbreaks]
