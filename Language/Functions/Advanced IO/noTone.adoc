= noTone()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
`tone()` 에 의해 트리거된 구형파의 생성을 멈춘다. tone 가 생성되지 않았을 때는 아무 효과 없다.

[%hardbreaks]


[float]
=== 문법
`noTone(pin)`


[float]
=== 매개변수
`pin`: tone 생성을 멈출 핀

[float]
=== 반환
없음

--
// OVERVIEW SECTION ENDS




// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== 주의와 경고
여러 핀에서 다른 피치를 연주하려면, 다음 핀에 `tone()` 을 부르기 전에  noTone() 을 부를 필요가 있다.

[%hardbreaks]
