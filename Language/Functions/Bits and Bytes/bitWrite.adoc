= bitWrite()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자 변수의 한 비트를 씁니다.

[%hardbreaks]


[float]
=== 문법
`bitWrite(x, n, b)`


[float]
=== 매개변수
`x`: 쓸 숫자 변수

`n`: 쓸 비트의 위치, LSB(맨 오른쪽 비트)가 0, 왼쪽으로 갈수록 1씩 증가

`b`: 비트에 쓸 값 (0 또는 1)

[float]
=== 반환
없음
