= bitWrite()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자 변수의 한 비트를 씀.

[%hardbreaks]


[float]
=== 문법
`bitWrite(x, n, b)`


[float]
=== 매개변수
`x`: 쓸 숫자

`n`: 쓸 비트, least-significant (가장 오른쪽) bit 0에서 시작

`b`: 비트에 쓸 값 (0 또는 1)

[float]
=== 반환
없음
