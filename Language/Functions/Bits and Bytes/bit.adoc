= bit()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
특정 비트(bit)의 값을 계산한다(bit 0 은 1, bit 1 은 2, bit 2 는 4, 등.).

[%hardbreaks]


[float]
=== 문법
`bit(n)`


[float]
=== 매개변수
`n`: 계산할 비트 값

[float]
=== 반환
비트 값.
