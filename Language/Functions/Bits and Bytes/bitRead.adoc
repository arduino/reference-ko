= bitRead()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자의 한 비트를 읽음.
[%hardbreaks]


[float]
=== 문법
`bitRead(x, n)`


[float]
=== 매개변수
`x`: 읽을 숫자

`n`: 읽을 비트, least-significant (가장 오른쪽) bit 0에서 시작


[float]
=== 반환
비트의 값 (0 또는 1).
