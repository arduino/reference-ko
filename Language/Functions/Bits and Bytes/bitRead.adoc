= bitRead()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자의 한 비트를 읽습니다.
[%hardbreaks]


[float]
=== 문법
`bitRead(x, n)`


[float]
=== 매개변수
`x`: 읽을 숫자

`n`: 읽을 비트, LSB(맨 오른쪽 비트)가 0, 왼쪽으로 갈수록 1씩 증가


[float]
=== 반환
비트의 값 (0 또는 1).
