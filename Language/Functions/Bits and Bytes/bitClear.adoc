= bitClear()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자 변수의 한 비트 지움(0을 씀).
[%hardbreaks]


[float]
=== 문법
`bitClear(x, n)`


[float]
=== 매개변수
`x`: 비트를 지울 숫자 변수

`n`: 지울 비트, 0에서 시작 least-significant (가장 오른쪽) bit

[float]
=== 반환
없음
