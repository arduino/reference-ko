= bitClear()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 설명
숫자 변수의 한 비트를 지웁니다(그 자리에 0을 씀).
[%hardbreaks]


[float]
=== 문법
`bitClear(x, n)`


[float]
=== 매개변수
`x`: 비트를 지울 숫자 변수

`n`: 지울 비트의 위치, LSB(맨 오른쪽 비트)가 0, 왼쪽으로 갈수록 1씩 증가

[float]
=== 반환
없음
